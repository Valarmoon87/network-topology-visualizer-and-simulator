# 🌐 Interactive Network Topology Visualizer and Simulator

## 📌 Overview
The **Interactive Network Topology Visualizer and Simulator** is a web-based application designed to help users create, visualize, and understand computer networks in a simple and interactive way.  

This system allows users to design network topologies using drag-and-drop functionality, simulate data flow, and receive intelligent suggestions for better network design.

---

## 🚀 Features

- 🖥️ Drag and drop network devices:
  - PC, Router, Switch, Server, Hub, Bridge, Access Point, Firewall, Modem  

- 🔗 Visual connection of devices  

- 🌐 Topology detection:
  - Star  
  - Bus  
  - Mesh  
  - Hybrid (Star + Bus, Mesh + Star, etc.)

- 📡 Data flow simulation  

- 🧠 AI-based suggestions and validation  

- ⚠️ Error detection for invalid connections  

- 📊 Real-time explanation of network behavior  

- 🎨 Clean and user-friendly white & green UI  

---

## 🏗️ System Architecture

The system consists of:

- **Frontend Interface** – User interaction and visualization  
- **Simulation Engine** – Handles topology logic and data flow  
- **AI Module** – Provides suggestions and detects errors  
- **Data Layer** – Stores devices and connections
